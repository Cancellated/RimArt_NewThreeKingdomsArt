  ── 画面描述（ArtDescriptionUtility_Global）──
  [style_group]       风格组（自动组合 style_clause + artextra_clause）
  [style_clause]      风格子句（如"作品象征着[ConceptAny]"）
  [artextra_clause]   画面细节子句（如"画面边缘由[subimageplural]组成"）
  [artconjunction]    风格过渡词（作为有趣的对比 / 顺带一提 / ...）
  [artstyle_adj]      艺术风格形容词（古典主义 / 印象派 / 巴洛克式 / ...共38种）
  [composition]       构图方法（三角形 / 矩形 / 动态 / 聚焦 / ...共10种）
  [symbolizes]        象征动词（象征着 / 似乎表达着 / 暗示着 / ...共10种）
  [depicts]           描绘动词（描绘 / 刻画 / 呈现 / ...共7种）
  [depiction]         描绘名词（描绘 / 刻画 / 表现 / ...共6种）
  [side_position]     位置（在背景中 / 在画面边缘 / 在主体附近 / ...共12种）
  [pawn_adj]          角色状态（死掉的 / 濒死的 / 身穿[Apparel]的 / 手持[Weapon]的 / ...共13种）
  [emotional_expression] 情绪（微笑着 / 皱着眉头 / 放声大笑 / 欣喜若狂 / ...共8种）
  [idles]             姿态动词（坐 / 站 / 飘浮 / 摆着造型 / ...共10种）
  [subimagesingle]    单个画面元素（一个[Character] / 一片[TerrainFeature] / ...）
  [subimageplural]    多个画面元素（[Quantity_adjphrase][Character] / ...）
  [subimageany]       随机单/多（→[subimagesingle] 或 [subimageplural]）

  ── 人物与生物（GlobalUtility）──
  [Character]         角色（→[PersonJob] 60% / [Animal] 40% / [Enemy] / [Mechanoid]）
  [PersonJob]         职业名（医生 / 商人 / 农夫 / ...外部词表）
  [Animal]            动物名（松鼠 / 爆炸鼠 / ...外部词表）
  [AnimalGroup]       动物量词（群 / 只 / ...外部词表）
  [Enemy]             敌人名（海盗 / ...外部词表）
  [Mechanoid]         机械体名（蜈蚣型 / ...外部词表）
  [NamePerson]        随机人名（游戏运行时生成）

  ── 物品与装备（GlobalUtility, 外部词表）──
  [Weapon]            武器名
  [Apparel]           服装名
  [Vegetable]         蔬菜名
  [Game]              游戏名
  [NaturalObject]     自然物名

  ── 地理与环境（GlobalUtility, 外部词表）──
  [Community]         社区类型（小镇 / 城市 / ...）
  [TerrainFeature]    地形特征（山脉 / 湖泊 / ...）
  [TreeType]          树木类型

  ── 概念与形容词（GlobalUtility, 外部词表）──
  [ConceptAny]        概念词（→[ConceptPositive] / [ConceptBadass] / [ConceptAngsty]）
  [ConceptPositive]   正面概念（希望 / 爱 / ...）
  [ConceptBadass]     硬核概念（力量 / 战争 / ...）
  [ConceptAngsty]     焦虑概念（孤独 / 绝望 / ...）
  [AdjectiveAny]      形容词（→[AdjectiveBadass] / [AdjectiveAngsty] / [AdjectiveFriendly] / [AdjectiveNatural]）
  [AdjectiveBadass]   硬核形容词
  [AdjectiveAngsty]   焦虑形容词
  [AdjectiveFriendly] 友好形容词
  [AdjectiveNatural]  自然形容词
  [AdjectiveLarge]    大型形容词

  ── 数量与形状（GlobalUtility）──
  [Quantity_adjphrase] 数量修饰（两个 / 三个 / 一群 / 数百 / 成千 / ...共17种）
  [Shape]             形状（三角形 / 方形 / 心形 / 星形 / 圆形 / ...共9种）
  [Color]             颜色名（外部词表）
  [ColorBadass]       硬核颜色名
  [Gore]              血腥描述（外部词表）

  ── 无故事专有（TalelessImages）──
  [image]             画面主体内容（一位孤独的[Character] / [ConceptAny]的抽象呈现 / ...）
  [desc_pawn]         人物姿态修饰（飞过天空 / 悬浮在空中 / 穿着[Apparel]骑在[Animal]上 / ...）

  ── 有故事专有（HasTale, 游戏运行时提供）──
  [tale_noun]         故事名称（如"X的死亡""Y与Z的战斗"）
  [DATE]              故事发生日期
  [desc_sentence]     故事描述句（由 Tale RulePack 提供）
  [circumstance_phrase] 故事情境短语（由 Tale RulePack 提供）

  ── 物理描述根（ArtDescription_Physical, 按艺术品类型）──
  [physical_description_root] 物理描述根（自动按类型选择）
    雕塑: "这件[sculpture][depicts]的是[image]。[desc_sentence_group]"
    武器: "这件武器上雕刻的内容[depicts]的是[image]。[desc_sentence_group]"
    家具: "这件家具上雕刻的内容[depiction]了[image]。[desc_sentence_group]"
    棺材: "这件棺材上雕刻的内容[depiction]了[image]。[desc_sentence_group]"
  [sculpture]         雕塑称呼（雕塑 / 作品 / 艺术品 / 雕像）
